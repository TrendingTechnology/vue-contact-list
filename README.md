# Vue Contact List

A contact list built with Vue.  Offers CRUD operations to a firebase database

### Demo
https://vue-contact-list-639f8.firebaseapp.com/


### Installing

+ Clone this repo 
+ `cd vue-contact-list`
+ run `yarn`
+ run `yarn serve`



## Built With

* [Vue.js](https://vuejs.org)
* [Firebase](https://firebase.google.com)
